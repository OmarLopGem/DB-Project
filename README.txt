This is the project presented by group 1

INSTRUCTIONS TO RUN PROJECT
1. Place the project folder anywhere you want
2. Inside the project with an integrated terminal (or any terminal inside the folder) write npm run dev
3. After you executed the command in the terminal, the terminal will point out that the project is up and running
4. Write on your explorer the followring url http://localhost:9090
5. You can create a new account or use an existing account

Notes:
-To simulate a real-life scenario we did not place an option to select the type of user
-The password for the users is similar to the email address (example. john@gmail.com -> john123, andy@gmail.com -> andy123, eli@gmail -> eli123)
-There is an admin user and the credentials for that one are
	-admin@gmail.com
	-admin123
-The wireframe is inside the wireframe folder inside the project folder
-The diagram is inside the diagram folder inside the project folder
-The JSON files are in the MongoDB Files folder inside the project folder
